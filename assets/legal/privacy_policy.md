# 韵算 (PoeMath) 隐私政策

**最后更新：2026 年 7 月 20 日**

欢迎使用 **韵算 (PoeMath)**。我们非常重视您及您孩子的隐私保护。本政策说明应用如何处理数据、权限与可选网络功能，与当前软件行为保持一致。

---

## 1. 产品原则

韵算是一款**离线优先**的儿童学习应用：

1. **核心学习本地完成**：诗词背诵、口算练习、公式浏览、打卡、成就、复习计划等，**不依赖网络**，飞行模式下可用。
2. **不向开发者服务器收集个人信息**：不采集姓名、年龄、学校、手机号、设备广告标识等用于画像或运营的数据。
3. **无广告、无内购、无分析/追踪 SDK**：不集成广告平台、行为分析或崩溃上报类数据收集服务。
4. **可选能力默认关闭或需主动触发**：备份、WebDAV 同步、检查更新、语音跟读、学习提醒等，仅在您（或家长）主动使用并授权时才会涉及相应权限或网络。

---

## 2. 我们不收集什么

韵算**不会**将下列信息上传到开发者可控的远程服务器：

- 姓名、年龄、性别、学校、年级等身份信息
- 手机号、邮箱、社交账号
- 精确位置、通讯录、相册内容（应用不申请这些权限）
- 用于广告的设备标识符、Cookie 行为追踪
- 将您的学习成绩、错题内容批量上报给开发方用于分析或广告

**说明：** 学习进度、错题、收藏等主要保存在**您的设备本地**；Android 系统可能按照您的系统设置执行备份或设备迁移。开发者无法在未经过您主动备份/同步的情况下远程查看这些数据，也无法访问 Android 系统保存的备份内容。

---

## 3. 本地数据存储

应用使用设备本地存储（含 Hive 数据库、应用私有目录）保存例如：

- 诗词学习进度、收藏、复习计划
- 口算练习记录、错题本、挑战记录
- 打卡、星星、等级、成就
- 主题、语速、音效/触觉开关等设置
- （若您配置）WebDAV 地址与账户信息的非密钥字段；密码等凭据使用系统安全存储加密保存

在 Android 设备上，操作系统可能按照设备厂商、系统账户及您的系统备份设置，将部分应用数据保存到系统云备份，或在换机时迁移到新设备。该过程由 Android 系统提供和控制，不经过韵算开发者服务器，开发者也无法查看这些系统备份内容。

**删除方式：**

- 卸载应用会删除当前设备上的应用本地数据（重要数据请先导出备份）
- 系统云备份或设备迁移中保存的副本可能在卸载后继续保留，并在重新安装或换机时恢复；请通过设备的系统备份设置管理或删除这些副本
- 可在系统设置中撤销麦克风、通知等权限
- 可通过「备份与恢复」先导出，再卸载或清空

## 4. 功能与数据处理对照

| 功能 | 是否需要网络 | 数据去向 | 触发方式 |
|------|-------------|---------|---------|
| 诗词 / 口算 / 公式 / 打卡 / 成就 | 否 | 仅本机 | 日常使用 |
| TTS 朗读 | 否* | 系统 TTS 引擎本地处理 | 您点击朗读 |
| 本地备份 / 恢复 | 否 | 导出 JSON 到本机目录或系统分享目标（由您选择） | 设置中主动操作 |
| Android 系统备份 / 设备迁移 | 视系统设置而定 | Android 系统备份服务或目标设备 | 由操作系统按您的系统设置执行 |
| WebDAV 云端同步 | 是 | **仅您填写的 WebDAV 服务器**（非韵算自有云） | 您配置并点击上传/下载 |
| 检查更新 / 安装 APK | 是 | 构建时配置的更新源（如下载 `latest.json` 与 APK） | 您点击「检查更新」且已配置更新地址 |
| 诗词跟读（语音识别） | 视系统而定 | 麦克风音频交给**系统语音识别服务**；应用用于比对诗句得分 | 您进入跟读并授权麦克风 |
| 每日学习提醒 | 否 | 本机定时通知 | 您开启提醒并授权通知 |

\* 部分系统 TTS 引擎由操作系统提供；韵算不把朗读文本发往开发者服务器。

### 4.1 本地备份 / 恢复

- 可将学习数据导出为 JSON 文件，或从 JSON 恢复
- 可通过系统「分享」发送到您选择的应用（邮件、网盘、文件管理器等）
- **不经过**韵算开发者服务器
- 请妥善保管备份文件，其中可能包含学习进度等个人使用记录

### 4.2 WebDAV 同步（可选）

- 默认关闭；需您自行填写服务器地址、路径、用户名与密码
- 上传/下载的是备份类数据，目标为**您指定的服务器**
- 凭据使用 `flutter_secure_storage` 等系统安全机制保存在本机
- 请仅使用您信任的 WebDAV 服务，并建议由家长协助配置

### 4.3 应用内更新（可选）

- 仅在构建时注入了更新检查地址（如 `UPDATE_CHECK_URL`）时可用
- 未配置时，「检查更新」入口不可用，不会发起更新请求
- 您点击检查后，应用可能下载版本信息与 APK，并在校验后请求安装
- 需要网络权限及（Android）安装应用包相关能力

### 4.4 诗词跟读

- 需要麦克风权限
- 使用系统语音识别；识别细节取决于手机厂商/系统实现
- 仅在您使用跟读功能时触发

### 4.5 本地通知

- 用于学习提醒等，在本机调度
- Android 8+ / 13+ 可能需要通知权限；开机完成后可能恢复已调度的提醒
- 不将通知内容上传至开发者服务器

### 4.6 Android 系统备份与设备迁移

- Android 可能默认为应用启用系统自动备份，并将部分应用私有数据保存到与设备关联的系统备份服务
- 换机、重新安装或设备迁移时，操作系统可能把备份数据恢复到新安装的韵算中
- 是否执行、备份范围、保留期限和删除方式取决于 Android 版本、设备厂商、系统账户及您的系统设置
- 这些系统备份不经过韵算开发者服务器，开发者无法访问；您可在设备的系统备份设置中查看、关闭或管理相关备份

## 5. Android 权限说明

以下权限与当前 `AndroidManifest` 声明一致，用途如下：

| 权限 | 用途 | 是否可拒绝 |
|------|------|-----------|
| `INTERNET` | WebDAV、检查更新等可选网络功能 | 系统级声明；不使用相关功能则不会主动联网学习 |
| `VIBRATE` | 答题/打卡等触觉反馈 | 可在应用设置中关闭触觉 |
| `REQUEST_INSTALL_PACKAGES` | 应用内下载 APK 后的安装流程 | 不使用检查更新则不会走到安装 |
| `RECORD_AUDIO` | 诗词跟读 | 可拒绝；拒绝后跟读不可用，其它功能不受影响 |
| `POST_NOTIFICATIONS` | 学习提醒通知 | 可拒绝；拒绝后无通知提醒 |
| `RECEIVE_BOOT_COMPLETED` | 开机后恢复已设置的本地定时提醒 | 仅服务通知调度 |

应用**不申请**通讯录、精准定位、短信、通话记录等与学习无关的权限。

## 6. 第三方服务与 SDK

- **无**广告 SDK、**无**统计分析 SDK、**无**商业崩溃收集 SDK（如面向运营的 Firebase Analytics / AdMob 等）
- 使用的开源库（如 Flutter 插件）用于本地存储、UI、TTS、文件选择、分享、HTTP 客户端等工程能力，**不以收集儿童个人信息为目的**
- 系统分享、文件选择、语音识别、TTS、系统备份与设备迁移属于**操作系统或您选择的目标应用**能力；其隐私实践以系统及对应应用的政策为准

---

## 7. 儿童隐私

韵算面向约 **6–12 岁** 学生，并遵循以下原则：

- 不要求注册账号，不建立面向儿童的社交关系
- 不展示广告，不提供内购
- 可选网络与敏感权限默认不强制；建议**家长协助**管理：
  - WebDAV 服务器与密码
  - 是否允许检查更新与安装 APK
  - 麦克风、通知权限
  - Android 系统备份与设备迁移设置
  - 备份文件的存放与分享对象
