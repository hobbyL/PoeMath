# 隐私政策

**最后更新：2026 年 7 月 12 日**

欢迎使用 **韵算 (PoeMath)**。我们非常重视您及您孩子的隐私保护。

## 1. 信息收集

**韵算不收集任何个人信息。**

- 不收集姓名、年龄、性别、学校等身份信息
- 不收集位置、设备标识符等设备信息
- 不收集学习记录、成绩等使用数据
- 不使用 Cookie 或任何追踪技术

## 2. 网络访问

**韵算完全离线运行。**

- 应用不发送任何网络请求
- 不连接任何远程服务器
- 不上传或下载任何数据
- 在飞行模式下所有功能完全可用

## 3. 数据存储

所有数据仅存储在用户设备本地：

- 学习进度、收藏、错题记录等均保存在设备本地存储
- 数据不会被上传到任何云端服务
- 卸载应用将永久删除所有本地数据
- 无数据备份/恢复功能（不涉及网络）

## 4. 第三方 SDK

**韵算不包含任何第三方数据收集 SDK。**

- 无广告 SDK（无 Google AdMob、Facebook Audience Network 等）
- 无分析 SDK（无 Firebase Analytics、Google Analytics 等）
- 无崩溃收集 SDK（无 Crashlytics、Sentry 等）
- 无社交分享 SDK

## 5. 内购与付费

**韵算不包含任何内购或付费功能。**

- 无订阅计划
- 无虚拟货币购买
- 无解锁付费内容
- 所有功能均免费使用

## 6. 儿童隐私

韵算专为 6-12 岁儿童设计，我们特别注意：

- 符合《儿童个人信息网络保护规定》要求
- 不收集任何儿童个人信息
- 不包含任何面向儿童的广告
- 不包含任何外部链接或可跳转的第三方页面
- 无社交功能、无用户间交互

## 7. 变更通知

如本隐私政策有任何变更，我们会在应用更新说明中通知用户。
