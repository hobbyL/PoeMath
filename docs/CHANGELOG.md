# 版本变更日志

所有重要变更记录在此文件中。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

## [1.0.0] - 2026-07-12

### 新增

#### 诗词模块
- 1000+ 首古诗词库（核心层 130 首 + 扩展层 400 首 + 探索层 500 首）
- 按年级/朝代/作者/主题标签浏览和搜索
- 诗词详情页：原文、拼音标注、注释、译文、赏析、创作背景
- 诗词背诵模式（填空/选择/渐进遮挡）
- 诗词收藏夹
- 艾宾浩斯间隔复习调度（间隔 1/3/7/14/30 天）
- 200+ 位诗人作者资料

#### 口算模块
- 12 类口算题型生成器：加减法/乘法口诀/有余除法/多位数乘除/混合运算/运算律/小数/分数/百分数/简易方程/比例/负数
- 1-6 年级 × 上/下学期 = 12 个学期精准课标配置
- 智能答题判定与即时反馈
- 分步骤解题讲解
- 6 类错因诊断规则：进位遗漏/退位遗漏/乘法口诀错误/运算顺序/余数错误/小数点对齐
- 错题本（收集/复查/标记已解决/删除）
- 练习会话统计与星星奖励

#### 公式知识库
- 60+ 条数学公式，按分类浏览
- 公式详情：参数说明、记忆技巧、例题
- 公式收藏与搜索
- 关联公式导航

#### 游戏化系统
- 每日打卡与连续天数统计
- 星星累积机制（答题正确率换算）
- 8 级等级体系：童生 → 秀才 → 举人 → 进士 → 探花 → 榜眼 → 状元 → 诗仙
- 成就勋章系统

#### 首页与个人中心
- 首页仪表盘：连续打卡、学习概览、快捷入口、今日目标
- 个人中心：学习统计面板、成就展示、主题切换、外观模式

#### 基础设施
- 双主题系统：国风诗词主题 / 童趣口算主题
- 5-Tab 底部导航（凹槽凸起设计）
- Hive 本地数据库（14 个 Box, 手写 TypeAdapter）
- Riverpod 状态管理
- GoRouter 路由体系
- 完全离线，无网络请求
- 无广告，无内购，无第三方追踪

### 安全
- 不收集任何个人信息
- 不包含任何广告或追踪 SDK
- 符合儿童隐私保护要求
