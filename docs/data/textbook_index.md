# 部编版小学语文古诗目录（1-6 年级）

> 数据来源：人民教育出版社 部编版语文教材 1-6 年级共 12 册
> 用途：核心层 130 首诗词的年级/学期/单元标注权威依据
> 权威 YAML：`tools/poem_importer/data/sources/textbook_index.yaml`

---

## 说明

- **必背 75 首**：教育部《义务教育语文课程标准（2022 年版）》附录 1 中"关于优秀诗文背诵推荐篇目"的小学阶段 75 首。
- **课内 112 首**：部编版 1-6 年级课本内出现的古诗（含"日积月累"、"课文"、"语文园地"等栏目）。
- **核心层 130 首**：以上两个集合的并集，去重后约 130 首（因大部分必背 75 首都在课内出现）。

---

## 数据字段

`textbook_index.yaml` 每首诗字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `id`             | string | 唯一 id，与 `poems_core.yaml` 对齐 |
| `title`          | string | 诗题 |
| `author`         | string | 作者 |
| `grade`          | int    | 年级 1-6 |
| `semester`       | string | 上 / 下 |
| `textbook_unit`  | string | "课文 X"、"日积月累"、"单元 X" 等 |
| `is_required_75` | bool   | 是否属于必背 75 首 |

---

## Phase 1 交付边界

- Phase 1 交付时的 `textbook_index.yaml` 收录了 30 首（1-6 年级各若干首，覆盖低/中/高年级）作为核心层的**权威索引示例**。
- 完整 130 首需在 Phase 1 后续迭代中人工补齐，本 Phase 主要交付：
  1. 索引 YAML 的 schema 与字段规范
  2. `import-core` 命令能按索引信息标注年级/学期/单元
  3. 30 首示例覆盖 1-6 年级全部 12 个学期至少 1 首

---

## 校对流程

核心层 130 首的每一首都需按下列步骤人工校对：

1. **原文**：逐字对照人教版教材本印刷版（避免网络传抄错字）。
2. **多音字拼音**：核心多音字建立 override 表（见 `pinyin_generator.dart`）：
   - `斜`（xiá）：山行、乌衣巷
   - `衰`（cuī）：回乡偶书
   - `骑`（jì）：过华清宫
   - `朝`（zhāo）：早发白帝城
   - `还`（huán）：早发白帝城
3. **年级/学期/单元**：以 `textbook_index.yaml` 为准。
4. **注释/译文/赏析**：择优选取适合小学生阅读的版本，简洁准确即可，不追求全面。
