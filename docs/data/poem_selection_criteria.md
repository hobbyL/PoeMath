# 诗词筛选标准（三层金字塔）

> PRD: `.trellis/tasks/07-12-phase-1-data-pipeline/prd.md` §R1.3
> 最后更新：2026-07-12

---

## 目标

从 22 万首古诗词大库（yxcs/poems-db）中筛选出 **1000 首**面向 6-12 岁儿童的
诗词学习清单，分三层：

| 层 | 数量 | 定位 | 筛选方式 |
|----|------|------|---------|
| 核心 | ~130 | 教材必学 + 课标必背 | 人工精校 |
| 扩展 | ~400 | 唐诗三百首 + 宋词精选 | 半自动 + 抽检 |
| 拓展 | ~500 | 主题标签精选 | 全自动 |

---

## 核心层 130 首（人工精校）

### 筛选口径

= 部编版 1-6 年级课本内的所有古诗（约 112 首）
∪ 义务教育课程标准（2022）附录 1 小学阶段必背 75 首
去重后约 130 首

### 硬性拒收条件

- 原文与人教版教材本存在字面差异且未标注（"版本差异"）
- 拼音校对未完成
- 无译文或译文不适合小学生（生僻词过多）
- 出处不明或作者存疑

### 通过标准

- 全部字段完整（`content` / `pinyin` / `translation` / `appreciation` / `annotations`）
- 年级 / 学期 / 单元 3 项标注全部存在
- `is_required` 已明确标注
- 多音字拼音已核对（对照 `pinyin_generator.dart` 的 override 表）
- 至少 1 条 `famous_lines`
- 至少 2 条 `tags`

---

## 扩展层 ~400 首（半自动 + 抽检）

### 筛选口径

- 唐诗三百首（清代蘅塘退士编选，311 首）
- 宋词精选（约 100 首，以《宋词三百首》为骨架）
- 去除已在核心层出现的重复条目

### 半自动处理

1. 通过 `import-extended` 从 YAML 源生成 JSON
2. 全部字段走 lpinyin 自动拼音生成
3. 注释/译文/赏析择优选取 yxcs 原库的一个版本

### 抽检要求

- 随机抽取 10-20%（即 40-80 首）人工校对
- 抽检重点：拼音多音字、注释准确性、译文可读性
- 抽检不合格率 >5% 则回炉重做整个批次

### 硬性拒收条件

- 拼音字数与原文字数不一致（校验器 warning）
- 与核心层内容重复
- 标签或作者字段为空

---

## 拓展层 ~500 首（全自动）

### 筛选口径

按主题标签精选，覆盖以下 10 类：

- 写景（山水/田园/四季）
- 送别
- 爱国
- 思乡
- 节日（春节/清明/端午/中秋/重阳/元宵）
- 哲理
- 咏物（梅/兰/竹/菊/松/柳）
- 农事
- 亲情
- 边塞

### 处理流程

1. 从 yxcs/poems-db 按 `tags` 字段筛选
2. 每个主题选取 40-60 首
3. 全脚本走 `import-explore`
4. 不做人工校对，靠自动校验器保底

### 通过标准

- 字段完整
- 通过 `validate` 命令（error = 0，warning ≤ 5%）
- 至少 1 个主题标签

---

## 通用字段完整性要求

所有三层诗词共同的字段完整性要求（`SchemaValidator` 校验）：

- `id` 全局唯一
- `title` / `author` / `dynasty` / `content` / `pinyin` / `layer` 必填
- `grade`（若存在）在 1-6 范围
- `content` 中的中文字符数 == `pinyin` 中的音节数（含声调）

---

## 数据体积上限

- 核心层：单首 ≤ 3KB（含赏析）
- 扩展层：单首 ≤ 2KB（简化赏析）
- 拓展层：单首 ≤ 1.5KB（仅关键信息）
- 总资产（含 authors / formulas）：≤ 3MB

---

## 边界情况处理

### 1. 同题多版本
- 优先选取教材版本
- 其次选取唐诗三百首收录版本
- 记录版本差异到 `annotations` 中

### 2. 作者存疑
- 优先采用教材注明的作者
- 若教材未涉及，采用《全唐诗》/《全宋词》的作者归属

### 3. 拼音古今音差异
- 一律采用**教材版本拼音**（例如 "斜" 在《山行》里教材注 xié，则用 xié；
  部分教材注古音 xiá，也可通过 override 覆盖）
- 若教材没有直接标注，采用主流学术观点

### 4. 生僻字
- 保留原字，`annotations` 中标注读音与含义
- 拼音字段中的音节由 `_charOverrides` 兜底
